# FrameLab

FrameLab is a desktop application for frame-accurate video inspection,
trimming, slow-motion export, and frame extraction.

## Current features

- Frame-by-frame video navigation
- Time-based navigation
- Start and stop frame marking
- Slow-motion clip export
- Individual frame export
- Batch frame export
- Monochrome image export
- Windows clipboard image copying
- All-intra-frame proxy generation for responsive seeking

## Requirements

- Python 3.11 or later
- Windows 10 or Windows 11
